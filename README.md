# 🌿 Жан мен Тән — Денсаулық Квизі

## Жоба туралы

**Жан мен Тән** сауықтыру орталығына арналған интерактивті денсаулық квизі. 7 сұрақтан тұрады, әр сұраққа сәйкес визуалды SVG суреттер бар. Соңында клиент жауаптарымен бірге WhatsApp-қа өтеді.

---

## 📁 Файл құрылымы

```
quiz-jan-men-tan.html   ← Толық квиз (бір файл, дайын)
```

## 🔧 Баптау

### WhatsApp нөмірін ауыстыру

`quiz-jan-men-tan.html` файлында **WHATSAPP_NUMBER** айнымалысын тап:

```javascript
const WHATSAPP_NUMBER = "77001234567"; // ← ОСЫ ЖЕРГЕ НӨМІРІҢДІ ЖАЗ
```

Мысалы: `"77071234567"` — елдің коды + нөмір, бос орынсыз, плюссыз.

## 🎨 Дизайн

| Элемент | Мән |
|---|---|
| Негізгі түс | `#3B6B4A` (жасыл — табиғи ем) |
| Акцент | `#C8956C` (жылы алтын) |
| Фон | `#E8E3DC` (жұмсақ крем) |
| Қаріп (заголовок) | Montserrat 700–900 |
| Қаріп (текст) | Noto Sans 400–600 |
| Стиль | Premium wellness / табиғи ем |

### Визуалды ерекшеліктер:
- Әр сұраққа **анимацияланған SVG сурет** (ауру белгілері, сағат, анализ т.б.)
- Опцияларда **emoji иконкалар** + hover эффекттер
- Прогресс бар (7 қадам)
- Бастапқы экран (Start screen)
- Нәтиже экраны (жауаптар қорытындысымен)
- WhatsApp батырмасы **пульсация анимациясымен**

## 📱 Квиз ағымы

```
[Бастапқы экран] → Тексеруді бастау
       ↓
[Сұрақ 1] Қай мәселе мазалайды? (9 опция)
       ↓
[Сұрақ 2] Қанша уақыттан бері? (5 опция)
       ↓
[Сұрақ 3] Қай белгі алаңдатады? (8 опция)
       ↓
[Сұрақ 4] Анализ тапсырдыңыз ба? (4 опция)
       ↓
[Сұрақ 5] Қандай ем қабылдайсыз? (4 опция)
       ↓
[Сұрақ 6] Емнің нәтижесі бар ма? (3 опция)
       ↓
[Сұрақ 7] Табиғи жолмен жазылғыңыз келе ме? (2 опция)
       ↓
[Нәтиже] → WhatsApp-қа жазу батырмасы
```

### Сұрақ қосу/өзгерту

```javascript

    // ...
  ],
  image: `<svg>...</svg>` // SVG сурет

```

### Түстерді өзгерту

```css

--bg: #E8E3DC;           /* фон */
```
